# @oh-my-github/ui

UI components for Oh My GitHub.

## Installation

```bash
npm install @oh-my-github/ui
```

## Usage

Import the CSS file in your `style.css` file.

```css
@import "tailwindcss";
@import "@oh-my-github/ui/style.css";
```

And then you can use the components in your Vue components.

```vue
<script setup>
import { Button } from '@oh-my-github/ui';
</script>

<template>
  <Button>Click me</Button>
</template>
```
